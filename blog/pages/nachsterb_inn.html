Instructions

This page serves as a testing space for the content loader system. You can add test text files and images to the /content/test/ directory and load them using the content hook attributes.

Example usage:

<div data-content-hook="my-content"
     data-content-path="../content/test/myfile.txt"
     data-content-type="text"></div>

Supported content types: text, html, markdown, image, video, youtube
