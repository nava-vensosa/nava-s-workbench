Safari Vim Navigator

Normal Mode
  • h/j/k/l - Navigate text
  • w/b/e - Word movement
  • {/} - Paragraph movement
  • gg/G - Top/Bottom
  • 0/$ - Line start/end
  • Enter - Click element
Free Mode (Caps Lock ON)
  • h/j/k/l - Move cursor (small steps)
  • u/i/o/p - Move cursor (large steps)
  • Enter - Click at cursor

Navigate any webpage with vim-style keyboard controls. Toggle Caps Lock for free cursor mode!
